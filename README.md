# mHealth Monitor Using Built-in Sensors

## Overview
This is an Android-based mobile health monitoring application that uses smartphone sensors to track real-time health and activity data.

## Features
- Heart rate estimation using camera (PPG)
- SpO₂ approximation
- Step counting and activity detection
- Stress level estimation using threshold logic
- Fall detection alerts
- Dashboard with real-time and historical data

## Tech Stack
- Android (Java/Kotlin)
- Sensor APIs
- SQLite
- Signal Processing
  

## Future Scope
- Integration with machine learning models
- Cloud-based data storage
